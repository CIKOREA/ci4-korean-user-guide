설치

CodeIgniter4는 직접 설치하거나, 컴포저를 이용하여 설치할 수 있습니다.

10.2.1 직접 설치

CodeIgniter설치 단계

  1. 패키지의 압축을 풉니 다.
  2. 서버에 CodeIgniter 폴더 및 파일을 업로드합니다. index.php 파일은 프로젝트 루트 내부의 public 폴더에있을 것입니다.
  3. 텍스트 편집기를 사용하여 application/Config/App.php파일을 열고 기본 URL을 설정합니다. 암호화나 세션을 사용하려면 암호화 키를 설정해야 합니다.
  4. 데이터베이스를 사용하려는 경우, 텍스트 편집기를 사용하여 application/Config/Database.php 파일을 열고 데이터베이스정보를 설정합니다.

보안 강화를 위해 Codeigniter의 system 및 application 디렉토리를 다른이름으로 변경할 수 있습니다. 이 경우 반드시 index.php 에서$ system_directory 와 $ application_directory 변수를 그에 맞게 설정해야 합니다.

여기에서 한발 더 나아가, 아예 해당 디렉토리들을 웹사이트 루트 보다 상위로 옮길 수 있습니다. 그러면 브라우저를 통한 직접 접근이 불가능 해 집니다. 기본적으로 직접 액세스를 방지하기 위해.htaccess 파일이 각 디렉토리에 포함되어 있지만 웹 서버의 설정에서 .htaccess 가 동작하지 않도록 할 수도 있으므로, 디렉토리를 아에 웹에서 접근할 수 없는 위치에 두는것이 가장 안전합니다.

이렇게 한후에 뷰 파일을 public에 두고 싶다면 view 디렉토리만 application에서 빼낼 수 있습니다. 그런다음에는 반드시 index.php파일을 열어 $view_folder변수를 설정해 주어야 합니다. 전체적으로 $system_path, $application_folder, $view_folder등 3가지 변수를 설정하게 되는 것이지요. 그리고 경로는 ‘/www/MyUser/system'와 같이 전체 경로를 적어주는것이 좋습니다.

실제 배포 환경에서는 반드시 PHP error reporting 등 개발시에만 필요한 것들을 비활성화 하는것도 중요합니다. CodeIgniter에서는 ENVIRONMENT 상수를 설정하면 됩니다. 보다 자세한 정보는 보안 페이지를 참조하세요 .
